## 📊 功能模块概览

### 1. 总览看板 (Dashboard) ✅

**访问路径**: `/reports/dashboard`

**核心功能**:
- 总发放额度统计
- 总充值额统计
- 总消费额统计
- 总退款额统计
- 净消费额计算
- 总交易笔数
- 参与者数量
- 摊位数量
- 活动筛选
- Excel 导出

**技术实现**:
- 后端 API: `GET /reports/summary`
- 前端组件: `Dashboard.tsx`
- 数据可视化: Ant Design Statistic 卡片

---

### 2. 摊位报表 (Booth Report) ✅

**访问路径**: `/reports/booths`

**核心功能**:
- 摊位维度数据统计
- 营业额、退款额、净收入
- 销量、成本、利润、利润率
- 表格排序
- 活动筛选
- Excel 导出

**技术实现**:
- 后端 API: `GET /reports/booths`
- 前端组件: `BoothReport.tsx`
- 数据展示: Ant Design Table

**统计指标**:
```
净收入 = 营业额 - 退款额
利润 = 净收入 - 总成本
利润率 = (利润 / 净收入) × 100%
```

---

### 3. 摊位排行榜 (Booth Leaderboard) ✅

**访问路径**: `/reports/booth-leaderboard`

**核心功能**:
- 营业额 TOP N 排行榜
- 利润 TOP N 排行榜
- 利润率 TOP N 排行榜
- 排行榜类型切换
- 显示数量选择 (TOP 10/20/50)
- 前三名特殊标识（金银铜牌）

**技术实现**:
- 后端 API:
  - `GET /leaderboard/revenue` - 营业额排行
  - `GET /leaderboard/profit` - 利润排行
  - `GET /leaderboard/roi` - 利润率排行
- 前端组件: `BoothLeaderboard.tsx`
- 可视化: 排名徽章、颜色编码

---

### 4. 商品排行榜 (Product Leaderboard) ✅

**访问路径**: `/reports/product-leaderboard`

**核心功能**:
- 销量 TOP N 排行榜
- 收入 TOP N 排行榜
- 利润 TOP N 排行榜
- 排序指标切换
- 显示数量选择
- 所属摊位信息展示

**技术实现**:
- 后端 API: `GET /leaderboard/products?metric=sales|revenue|profit`
- 前端组件: `ProductLeaderboard.tsx`
- 数据展示: 排名表格 + 摊位关联

---

### 5. 异常审计报表 (Audit Logs) ✅

**访问路径**: `/reports/audit-logs`

**核心功能**:
- 高频退款检测（1小时内超过5次）
- 大额更正检测（金额超过1000元）
- 可疑操作检测（深夜交易：22:00-06:00）
- 异常类型筛选
- 活动筛选
- 显示数量选择

**技术实现**:
- 后端 API: `GET /reports/audit-logs?flag_type=all|high_frequency_refund|large_adjustment|suspicious_operation`
- 前端组件: `AuditLogs.tsx`
- 异常标记: 图标 + 颜色编码

**检测规则**:
| 异常类型 | 检测条件 | 标记颜色 |
|---------|---------|---------|
| 高频退款 | 同一操作员1小时内退款>5次 | 橙色 |
| 大额更正 | adjust 类型且金额>1000元 | 红色 |
| 可疑操作 | 深夜时段（22:00-06:00）交易 | 紫色 |

---

### 6. 报表导出 (Export) ✅

**访问路径**: `/reports/export`

**核心功能**:
- 总览统计报表导出
- 摊位报表导出
- 商品报表导出
- 交易流水报表导出（最多10000条）
- 活动筛选
- Excel 格式
- 自动文件命名

**技术实现**:
- 后端 API: `GET /reports/export/excel?report_type=summary|booths|products|transactions`
- 前端组件: `ExportPage.tsx`
- 导出服务: `export_service.py` (使用 openpyxl)

**文件命名格式**:
```
{报表类型}_{活动名称}_{日期}.xlsx
例如: 摊位报表_春季园游会_2026-05-09.xlsx
```

## 📊 数据统计说明

### 统计基础

所有统计基于 **交易流水表（transactions）** 的 append-only 模式：

✅ **优势**:
- 完整的历史记录
- 支持审计追溯
- 数据一致性保证
- 支持任意时间段统计

### 交易类型映射

| 交易类型 | 说明 | 统计影响 |
|---------|------|----------|
| `issue` | 发卡 | 总发放额度 |
| `recharge` | 充值 | 总充值额 |
| `pay` | 支付 | 总消费额、摊位营业额 |
| `refund` | 退款 | 总退款额 |
| `adjust` | 调整 | 异常审计（大额更正） |
| `void` | 作废 | 不计入常规统计 |
| `expire` | 过期 | 不计入常规统计 |

## 📝 使用指南

### 快速开始

1. **登录系统**
   - 使用具有报表权限的账号登录

2. **访问报表中心**
   - 点击左侧菜单 "报表中心"
   - 展开子菜单查看各功能模块

3. **查看总览看板**
   - 选择活动（可选）
   - 查看关键统计指标
   - 导出 Excel 报表

4. **查看排行榜**
   - 选择排行榜类型
   - 选择显示数量
   - 查看排名和数据

5. **查看异常审计**
   - 选择异常类型
   - 查看异常交易记录
   - 分析异常原因

6. **导出报表**
   - 选择报表类型
   - 选择活动（可选）
   - 点击导出按钮

**场景1: 活动结束后统计总体数据**
1. 进入"总览看板"

**场景2: 查看摊位经营情况**
1. 进入"摊位报表"
2. 选择活动
3. 按利润率排序

**场景3: 评选优秀摊位**
1. 进入"摊位排行榜"
2. 切换不同排行榜类型
3. 查看 TOP 10

**场景4: 审计异常交易**
