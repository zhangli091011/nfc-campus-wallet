# 🚀 快速开始 - 报表中心使用指南

## ⚡ 立即开始 (3步)

### 1️⃣ 重启后端服务

```powershell
# 停止并重启
Get-Process python | Stop-Process -Force
python start_server.py
```

### 2️⃣ 验证CORS修复

```bash
python test_cors.py
```

**期望看到**:
```
✅ 所有CORS头检查通过
✅ 报表端点CORS头存在
✅ OPTIONS预检请求成功
```

### 3️⃣ 刷新前端

1. 打开浏览器 http://localhost:3000
2. 按 `Ctrl+Shift+R` (硬刷新)
3. 点击左侧菜单 "报表中心"

---

## 📊 报表中心功能

### 总览看板
**路径**: 报表中心 → 总览看板

**功能**:
- 查看8个关键统计指标
- 选择活动筛选数据
- 导出Excel报表

**使用**:
1. 选择活动(可选)
2. 查看统计数据
3. 点击"导出Excel"下载报表

---

### 摊位报表
**路径**: 报表中心 → 摊位报表

**功能**:
- 查看所有摊位经营数据
- 营业额、利润、利润率等
- 表格排序和筛选

**使用**:
1. 选择活动(可选)
2. 点击表头排序
3. 查看摊位详细数据
4. 导出Excel报表

---

### 摊位排行榜
**路径**: 报表中心 → 摊位排行榜

**功能**:
- 营业额TOP N
- 利润TOP N
- 利润率TOP N

**使用**:
1. 选择活动
2. 选择排行榜类型(营业额/利润/利润率)
3. 选择显示数量(TOP 10/20/50)
4. 查看排名

**特色**:
- 🥇 第一名金色背景
- 🥈 第二名银色背景
- 🥉 第三名铜色背景

---

### 商品排行榜
**路径**: 报表中心 → 商品排行榜

**功能**:
- 销量TOP N
- 收入TOP N
- 利润TOP N

**使用**:
1. 选择活动
2. 选择排序指标(销量/收入/利润)
3. 选择显示数量
4. 查看商品排名

---

### 异常审计
**路径**: 报表中心 → 异常审计

**功能**:
- 高频退款检测
- 大额更正检测
- 可疑操作检测

**使用**:
1. 选择活动
2. 选择异常类型
3. 选择显示数量
4. 查看异常记录

**异常类型**:
- 🟠 高频退款: 1小时内超过5次
- 🔴 大额更正: 金额超过1000元
- 🟣 可疑操作: 深夜交易(22:00-06:00)

---

### 报表导出
**路径**: 报表中心 → 报表导出

**功能**:
- 导出总览统计
- 导出摊位报表
- 导出商品报表
- 导出交易流水

**使用**:
1. 选择活动(可选)
2. 点击对应报表的"导出Excel"
3. 文件自动下载

**文件命名**:
```
{报表类型}_{活动名称}_{日期}.xlsx
例如: 摊位报表_春季园游会_2026-05-09.xlsx
```

## 🔧 故障排除

### 问题1: 看不到报表中心菜单

**解决**:
1. 确认已登录
2. 确认用户角色(需要 super_admin/event_admin/reviewer)
3. 刷新页面

### 问题2: 数据加载失败

**解决**:
1. 检查后端服务是否运行
   ```bash
   curl http://localhost:8000/health
   ```
2. 检查浏览器控制台错误
3. 清除缓存并刷新

### 问题3: CORS错误

**解决**:
1. 重启后端服务
   ```powershell
   Get-Process python | Stop-Process -Force
   python start_server.py
   ```
2. 运行CORS测试
   ```bash
   python test_cors.py
   ```
3. 清除浏览器缓存
4. 硬刷新页面 (Ctrl+Shift+R)

### 问题4: 导出失败

**解决**:
1. 检查是否已登录
2. 检查网络连接
3. 查看浏览器控制台错误
4. 尝试导出其他报表
