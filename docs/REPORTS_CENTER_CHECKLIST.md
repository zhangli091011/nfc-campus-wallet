3. **可疑操作**: 深夜时段（22:00-06:00）的交易

## 📊 数据统计说明

### 统计基础

所有统计数据基于 **交易流水表（transactions）** 的 append-only 模式：

- ✅ 不依赖账户余额字段
- ✅ 支持历史数据追溯
- ✅ 保证数据一致性
- ✅ 支持审计和对账

### 交易类型

| 类型 | 说明 | 统计影响 |
|------|------|----------|
| `issue` | 发卡 | 计入总发放额度 |
| `recharge` | 充值 | 计入总充值额 |
| `pay` | 支付 | 计入总消费额、摊位营业额 |
| `refund` | 退款 | 计入总退款额 |
| `adjust` | 调整 | 不计入常规统计 |
| `void` | 作废 | 不计入常规统计 |
| `expire` | 过期 | 不计入常规统计 |

### 计算公式

**净消费额**:
```
净消费额 = 总消费额 - 总退款额
```

**摊位净收入**:
```
净收入 = 营业额 - 退款额
```

**摊位利润**:
```
利润 = 净收入 - 总成本
```

**利润率**:
```
利润率 = (利润 / 净收入) × 100%
```

## 📝 使用说明

### 1. 访问报表中心

登录后，在左侧菜单点击 **"报表中心"** 展开子菜单。

### 2. 查看总览看板

1. 点击 **"总览看板"**
2. 选择活动（可选，不选则显示全部活动数据）
3. 查看关键指标统计
4. 点击 **"导出 Excel"** 下载报表

### 3. 查看摊位报表

1. 点击 **"摊位报表"**
2. 选择活动筛选
3. 点击表头进行排序
4. 查看各摊位经营数据
5. 导出 Excel 进行进一步分析

### 4. 查看排行榜

1. 点击 **"摊位排行榜"** 或 **"商品排行榜"**
2. 选择活动
3. 切换排行榜类型（营业额/利润/利润率 或 销量/收入/利润）
4. 选择显示数量（TOP 10/20/50）
5. 查看排名和数据

### 5. 查看异常审计

1. 点击 **"异常审计"**
2. 选择活动和异常类型
3. 查看异常交易记录
4. 分析异常原因
5. 必要时采取措施

### 6. 批量导出报表

1. 点击 **"报表导出"**
2. 选择活动（可选）
3. 点击对应报表的 **"导出 Excel"** 按钮
4. 文件自动下载到本地
