已修复安卓端金额计算错误：
- ❌ **修复前**: 111元商品扣款1.11元，200元充值显示2元

## 常见问题

### Q1: 修复后还是显示错误金额？
A: 请确保：
1. 已重新编译APK
2. 已卸载旧版本并安装新版本
3. 已重启应用

### Q2: 后台显示正确但安卓端显示错误？
A: 这可能是缓存问题，请：
1. 清除应用数据
2. 重新登录
3. 重新刷卡查询余额

### Q3: 如何确认使用的是新版本？
A: 查看编译时间：
```bash
ls -lh android/app/build/outputs/apk/debug/app-debug.apk
```

## 修复的技术细节

### 问题根源
```java
// 错误代码（已修复）
for (CartItem item : cartItems) {
    totalCents += item.getTotalPrice();  // 返回元（double），但累加到分变量
}
// 结果：111元商品被当作111分 = 1.11元
```

### 修复方案
```java
// 正确代码
for (CartItem item : cartItems) {
    totalCents += item.getTotalPriceInCents();  // 返回分（int）
}
// 结果：111元商品 = 11100分，正确！
```
